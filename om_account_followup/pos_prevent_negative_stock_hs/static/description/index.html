Prevent Negative Stock in POS

Ensure inventory accuracy by blocking POS orders with insufficient stock.

Features

  • Blocks POS orders for out-of-stock items
  • Displays user-friendly warning messages
  • Works seamlessly in real-time
  • Keeps inventory accurate and prevents overselling

Installation

For Odoo Community/Enterprise (On-Premise)

  1. Place the module in your Odoo custom addons folder
  2. Update app list and install the module
  3. Use POS as usual – the system will block negative stock orders

For Odoo Online (SaaS)

  1. Download the module zip file
  2. Enable Debug Mode (Add ?debug=1 to your URL)
  3. Go to Apps menu → Click "Import Module"
  4. Select the downloaded zip file and upload
  5. Click "Install" and start using the module

For Odoo.sh

  1. Connect to your Odoo.sh repository via Git
  2. Add the module to your custom addons directory
  3. Push changes to your repository
  4. Update app list and install the module
  5. Use POS as usual – the system will block negative stock orders

Configuration

  • No additional configuration required
  • Works automatically for all consumable products
  • Affects POS orders only (not Sales Orders)
  • Checks real-time stock availability including items in cart

Requirements

  • Odoo 16.0+
  • Odoo 17.0+
  • Odoo 18.0+
  • Point of Sale module installed
  • Stock/Inventory module installed

Compatibility

Fully compatible with:

  • Odoo Online (SaaS)
  • Odoo.sh
  • Odoo Community
  • Odoo Enterprise
