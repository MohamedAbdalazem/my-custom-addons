Usage
=====

To use this module, you need to:

1. Go to Invoicing > Configuration > Reconciliation Models.
2. In the "Invoices Matching Rule" model, disable "Partner Is Set & Matches" option.
3. In the bank statement, when it is posted, an Internal Transfer Button will appear in the lines.
4. Use the wizard to select the destination journal and the payment method.
5. An Internal Transfer will be created for the statement line.
6. Use the reconciliation widget for reconcile the statement line and the internal transfer (they both have the same reference).
